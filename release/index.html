ENIGMA Ontology

In this page you will find the detailed the documentation of all classes, properties, data properties and individuals present in the different modules of the ENIGMA ontology.


Ontology Serialization License Description
The Cohort Ontology Turtle CC-BY 2.0. An ontology to describe cohorts of patients participating in ENIGMA projects
The Organization Ontology Turtle CC-BY 2.0. An ontology representing organizations that participate in ENIGMA consortium
The Person Ontology Turtle CC-BY 2.0. An ontology representing the scientists that participate in ENIGMA projects and their roles
The Project Ontology Turtle CC-BY 2.0. An ontology representing projects of ENIGMA
The Scanner Ontology Turtle CC-BY 2.0. An ontology representing scanners and instruments used to make the measurements in ENIGMA
The Core Upper Level Ontology Turtle CC-BY 2.0. An ontology representing main concepts of ENIGMA
The Working Group Ontology Turtle CC-BY 2.0. An ontology representing Working Groups that participate in ENIGMA consortium
