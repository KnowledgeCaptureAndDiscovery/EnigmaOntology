has statusdp back to ToC or Data Property ToC

IRI: https://w3id.org/enigma#hasStatus

Describes the current activity status of a person at a given organization. "Active" specifies that one is currently active within an organization. "Inactive" specifies that one is currently still part of an organization but not currently active. "Left Organization" specifies that one is no longer a part of an organization.
Example
JohnSmith hasQualifiedAffiliationRelationship "QualifiedAffiliation" hasStatus "Inactive"
has domain
Qualified Affiliation c
has range
{ "Active" , "Inactive" , "Left Organization" }
