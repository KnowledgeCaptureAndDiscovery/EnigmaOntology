ENIGMA Ontology

In this page you will find the detailed the documentation of all classes, properties, data properties and individuals present in the different modules of the ENIGMA ontology.


Ontology Serialization License Description
The Enigma Core Ontology Turtle CC-BY 2.0. Represent main concepts of the ENIGMA consortium and its datasets
The Cohort Ontology Turtle CC-BY 2.0. Describe cohorts of patients
The Organization Ontology Turtle CC-BY 2.0. Represent organizations that participate in the ENIGMA consortium
The Person Ontology Turtle CC-BY 2.0. Represent the scientists that participate in ENIGMA working groups and projects
The Project Ontology Turtle CC-BY 2.0. Represent projects undertaken by ENIGMA working groups
The Working Group Ontology Turtle CC-BY 2.0. Represent working groups that participate in the ENIGMA consortium
The Roles Ontology Turtle CC-BY 2.0. Represent roles of persons in ENIGMA projects, organizations, papers or working groups
The Demographic Ontology Turtle CC-BY 2.0. Represent demographics for cohort datasets
The Disorder Ontology Turtle CC-BY 2.0. Represent mental health disorders for cohort datasets
The Medication Ontology Turtle CC-BY 2.0. Represent medications for cohort datasets
The Psychopathology Ontology Turtle CC-BY 2.0. Represent psychopathology information for cohort datasets
The Neuropsychiatric Ontology Turtle CC-BY 2.0. Represent neuropsychiatric information for cohort datasets
The Scores Ontology Turtle CC-BY 2.0. Represent questionnaire and test scores for cohort datasets
